First-time setup instructions
  1. Create a free account at ngrok.com
  2. Copy your authtoken from the ngrok dashboard
  3. Paste it above and click Start Tunnel
  4. Click Register in MT to auto-create the webhook endpoint (requires API key from the Setup page)
  5. Optional: Claim a free static domain so the URL never changes
Orgs, runs, and signing
  • The banner below the panel checks whether your Setup org’s MT webhook URL matches this listener.
  • Multiple orgs can use the same listener URL if each registers a webhook to it.
  • Events match runs using resource IDs from runs/ manifests; other IDs stay unmatched.
  • One DATALOADER_WEBHOOK_SECRET at a time — for several orgs on one URL, leave it empty in sandbox or run one org per instance.
